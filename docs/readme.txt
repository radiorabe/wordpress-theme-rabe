RaBe WordPress theme
-----------------------
RaBe is is a omega child wordpress theme built for Radio Bern (RaBe).

Created by Sebastian Liechti, Marius Glauser https://momou.ch

Install
-------
1. Download "omega" parent theme from http://wordpress.org/themes/omega
2. Go to your WordPress dashboard and select Appearance > Install, click Upload link.
3. Upload both Omega and this child theme, then activate the child theme
4. Required: The theme needs several plugins for its correct function
	- Event Organiser (https://wordpress.org/plugins/event-organiser/)
	- Restrict Taxonomies
	  (Adapted version from https://github.com/brknkfr/Restrict-Taxonomies)
	- Fast Secure Contact Form
	  (https://wordpress.org/plugins/si-contact-form/)
5. Recommended:
	- Polylang
	  (https://de.wordpress.org/plugins/polylang/)
	- Simple Colorbox
	  (https://de.wordpress.org/plugins/simple-colorbox/)
6. Optional:
	- Email Address Encoder
	  (https://wordpress.org/plugins/email-address-encoder/)
	- Import users from CSV with meta
	  (https://wordpress.org/support/plugin/import-users-from-csv-with-meta)


Customization
-------------

For correct permission settings of the users, some modifications on the 
"Restrict Taxonomies" plugin was needed. That's the reason, this plugin have to 
be installed from the github repo.
